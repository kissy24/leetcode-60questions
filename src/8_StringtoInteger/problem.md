# myAtoi関数の実装

文字列を32ビット符号付き整数に変換する `myAtoi(string s)` 関数を実装してください。

`myAtoi(string s)` のアルゴリズムは以下の通りです：

1. **空白文字**: 先頭の空白文字（`" "`）を無視します。
2. **符号**: 次の文字が `'-'` または `'+'` かどうかをチェックして符号を決定し、どちらでもない場合は正の数と仮定します。
3. **変換**: 先頭のゼロをスキップして、数字以外の文字が現れるか文字列の末尾に達するまで整数を読み取ります。数字が読み取られなかった場合、結果は0です。
4. **丸め**: 整数が32ビット符号付き整数の範囲 `[-2³¹, 2³¹ - 1]` を超えている場合、整数を範囲内に収まるように丸めます。具体的には、`-2³¹` より小さい整数は `-2³¹` に、`2³¹ - 1` より大きい整数は `2³¹ - 1` に丸めます。

最終結果として整数を返します。

## 例

**例1:**
- **入力:** s = "42"
- **出力:** 42
- **説明:**

```
下線の文字は読み取られた文字で、キャレットは現在の読み取り位置です。
ステップ1: "42" (先頭の空白がないため、文字は読み取られない)
         ^
ステップ2: "42" ('-'も'+'もないため、文字は読み取られない)
         ^
ステップ3: "42" ("42"が読み取られる)
           ^
```

**例2:**
- **入力:** s = " -042"
- **出力:** -42
- **説明:**

```
ステップ1: "   -042" (先頭の空白が読み取られて無視される)
            ^
ステップ2: "   -042" ('-'が読み取られるので、結果は負の数になる)
             ^
ステップ3: "   -042" ("042"が読み取られ、結果では先頭のゼロが無視される)
               ^
```

**例3:**
- **入力:** s = "1337c0d3"
- **出力:** 1337
- **説明:**

```
ステップ1: "1337c0d3" (先頭の空白がないため、文字は読み取られない)
         ^
ステップ2: "1337c0d3" ('-'も'+'もないため、文字は読み取られない)
         ^
ステップ3: "1337c0d3" ("1337"が読み取られ、次の文字が数字以外なので読み取りが停止)
             ^
```

**例4:**
- **入力:** s = "0-1"
- **出力:** 0
- **説明:**

```
ステップ1: "0-1" (先頭の空白がないため、文字は読み取られない)
         ^
ステップ2: "0-1" ('-'も'+'もないため、文字は読み取られない)
         ^
ステップ3: "0-1" ("0"が読み取られ、次の文字が数字以外なので読み取りが停止)
          ^
```

**例5:**
- **入力:** s = "words and 987"
- **出力:** 0
- **説明:**
最初の数字以外の文字 'w' で読み取りが停止します。

## 制約

- `0 <= s.length <= 200`
- `s` は英字（大文字・小文字）、数字（`0-9`）、`' '`、`'+'`、`'-'`、`'.'` で構成されます。
